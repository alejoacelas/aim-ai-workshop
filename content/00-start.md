# Coding agents 101

By the end of this session you'll have a coding agent running on your computer, connected to your Google Workspace, and you'll have built something cool with it.

Most people who try this are surprised by how much they can get done without knowing how to code. The goal isn't to learn programming — it's to get a capable agent working for you today.

## Session overview

Go through the tasks below in order. Each one builds on the last.

0. [Install your AI agent](#install) — get Claude Code or Codex running on your machine
1. [Clean up your Downloads folder](#clean-downloads) — your first agentic task
2. [Try using skills](#skills) — give your agent reusable expertise
3. [Connect your Google Workspace](#google-workspace) — read email, edit Sheets, schedule meetings
4. [Build something useful](#build-something) — pick a real project and ship it
